## What This Is

This project is an interactive, web-based learning platform for DJ fundamentals.

It teaches DJing by:
- progressively unlocking vocabulary and controls
- embedding lessons directly into interaction
- using a minimalist DJ tool whose layout mirrors real-world hardware conventions

The platform is designed so that learning transfers naturally from the screen to physical DJ equipment, without explicit instruction.

This is not professional DJ software.
This is a learning instrument.

---

## Core Philosophy

1. **Implicit learning over explanation**
   - The UI teaches through structure and affordance, not lists or manuals

2. **Progressive disclosure**
   - Users are never shown more controls than they need at a given moment

3. **Vocabulary as capability**
   - Unlocking a concept unlocks a word *and* a control

4. **Nonlinear exploration**
   - Learning is organized as a skill tree, not a linear course

5. **Minimal DSP**
   - Early learning focuses on timing, structure, and intent, not sound manipulation

6. **Spatial transfer**
   - The relative placement of controls reflects common DJ hardware layouts so users build real-world intuition automatically

---

## Target Audience

Primary:
- Absolute beginners
- Programmers and technical learners
- Overwhelm-sensitive users

Secondary:
- Educators
- Hobbyist DJs interested in fundamentals
- Users who want confidence before touching real gear

---

## Mental Model

DJing is framed as:
- two independent playback systems
- each with its own clock
- blended intentionally over time

The platform exposes this model through interaction, not abstraction.

---

## Core Components

### 1. Skill Tree Interface
- Nodes represent concepts
- Edges represent conceptual dependencies
- Unlocking a node may:
  - introduce new vocabulary
  - reveal a new control
  - enable a new interaction

### 2. Minimalist DJ Tool
- Web-based
- Feature-gated
- Layout-constrained to reflect real hardware conventions

### 3. Lesson Panels
- Short, contextual explanations
- Directly tied to active controls
- No long-form tutorials or manuals

---

## What Success Looks Like

A new user can:
- explain what a track, deck, and crossfader are
- beatmatch two tracks by ear
- transition between tracks intentionally
- walk up to unfamiliar DJ hardware and orient themselves confidently

All without:
- memorizing jargon
- watching long videos
- buying equipment
- feeling embarrassed or out of place
